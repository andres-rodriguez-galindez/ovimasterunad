Objetivos del OVI


1. Aprender y desarrollar el conocimiento en la utilización de etiquetas HTML5 Y CSS3
2. Desarrollar investigación propias de otras etiquetas e implementar en el curso de Diseño WEB
3. Identificar las clases y Id de para un diseño correcto en CSS3
4. Aprender cada uno de los elementos semánticos básico para una pagina de desarrollo web
5. Aprender a enlazar plugins de redes sociales en paginas web

Objetivos del curso

1. Fundamentarse teóricamente sobre las generalidades básicas de Diseño Web
2. Que el estudiante pueda utilizar de forma correcta los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de la construcción de sitios Web
3. Que el estudiante implemente servicios y sitios web a través de los sistemas gestores CMS Y LMS libres mas usados en la actualidad.
